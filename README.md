Linux Process Management Commands
=================================

top, ps, jobs, kill 명령어 정리


1. top
---------------------------------
시스템의 전체 프로세스 및 리소스 상태를 실시간 모니터링하는 명령어.

[주요 기능]
- CPU / 메모리 사용량 확인
- 프로세스별 자원 사용량 표시
- 실시간 업데이트 (기본 3초)

[주요 단축키]
q : 종료
k : 프로세스 종료
h : 도움말
M : 메모리 사용량 기준 정렬
P : CPU 사용량 기준 정렬

[사용 예시]
top
top -p 1234   (특정 PID만 모니터링)


2. ps
---------------------------------
현재 실행 중인 프로세스를 출력하는 명령어.

[주요 옵션]
ps         : 현재 터미널의 프로세스 출력
ps -e      : 모든 프로세스 출력
ps -f      : 풀 포맷 출력
ps aux     : 시스템 전체 프로세스 상세 보기

[사용 예시]
ps
ps -ef
ps aux | grep python


3. jobs
---------------------------------
현재 셸에서 백그라운드 실행 중인 작업 목록 출력.

[주요 상태]
Running : 실행 중
Stopped : 일시 중지
Done    : 종료됨

[사용 예시]
command &     (백그라운드 실행)
jobs          (작업 목록 확인)
fg %1         (1번 작업 포어그라운드)
bg %1         (1번 작업 백그라운드)


4. kill
---------------------------------
프로세스에 신호(signal)를 보내 종료하거나 동작을 제어하는 명령어.

[주요 신호]
1  : SIGHUP   (재시작)
9  : SIGKILL  (강제 종료)
15 : SIGTERM  (정상 종료 요청, 기본)

[사용 예시]
kill 1234        (기본 SIGTERM)
kill -9 1234     (강제 종료)
kill -HUP 1234   (재로드)


요약표
---------------------------------
- top   : 시스템 상태 실시간 모니터링
- ps    : 실행 중인 프로세스 목록 출력
- jobs  : 셸에서 실행 중인 작업 관리
- kill  : 프로세스 종료 또는 신호 전송
